# PDF EDITOR v0.1.0

First public release of PDF EDITOR for macOS.

## Included

- Interactive PDF splitting workflow in Terminal
- CSV and XLSX input support
- Filename suffix input
- Automatic output folder creation
- Automatic duplicate filename handling
- Text report generation
- Double-click launcher files for non-technical users
- Release zip generator for GitHub Releases

## End-User Setup

1. Download `PDF Editor.zip` from the GitHub Release page
2. Extract the zip file
3. Double-click `Setup PDF Editor.command`
4. After setup finishes, double-click `Launch PDF Editor.command`

## Notes

- macOS may ask you to confirm opening `.command` files the first time.
- The first setup creates a local virtual environment and installs the required dependency.
- This release is intended for macOS use.
